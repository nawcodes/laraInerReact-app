
## About LaraInnerReact App

LaraInnerReact App it its aplication base with Laravel 10 | Inertia JS | React JS | This is simple crud app or simple todo app 

## Feature
- Create 
- Edit 
- Delete

## How To Install 
1. clone this repo 
2. composer i
3. npm i
4. php artisan migrate --seed
5. php artisan serve
6. npm run dev
